# CPU使用率计算模块

cpu使用率是根据时间段计算的，没有办法统计某一时刻的使用率。

运行cup-usage。

```bash
./cpu-usage
```

再开一个终端，并且在0核的做一些消耗cpu的事情。

```bash
taskset -c 0 g++ -O3 ./src/cpu/usage-test.cc ./src/cpu/usage.cc -o ./bin/cpu-usage
```
